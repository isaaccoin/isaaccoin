What is Isaaccoin?
----------------

Isaaccoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 5 minute block targets
 - subsidy halves in 100k blocks 
 - 350000 total coins

The rest is the same as Bitcoin.
 - 20 coins per block
 - 12 blocks to retarget difficulty
